Python поддерживает все распространенные арифметические операции:
*  "+" сумма
*  "-" разность
*  "*" произведение
*  / деление с остатком
*  // деление без остатка
*  ** возведение в степень
*  % получение остатка от деления
```
Пример кода:
# Сумма
print(1+1)
# Разница
print(1-1)
# Умножение
print(2*2)
# Деление с остатком (float)
print(3/2)
# Деление без остатка
print(5//2)
# Возведение в степень (36)
print(6**2)
# Остаток от деления
print(5 % 2)
```
